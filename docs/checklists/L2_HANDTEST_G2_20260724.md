## 关键发现

### ✅ EP3 (PR #80) 在 G2 跨片验证通过

`match_summary.source_counts.embedding_topk = 18` + `topk.topk_count = 18` — Top-K 重排在新片（4.45 GB）上 100% 路径命中，无任何 top-1 退化或纯 heuristic fallback。**EP3 实现端到端打通跨片稳定性可证**。

### ✅ L2 exit §1「≥2 样片」达成

| 样片 | 文件大小 | detect_scenes 总耗时 | qa_output | embedding_topk |
|---|---|---|---|---|
| G1 飞驰人生3 (Historical War) | 1.55 GB | ~10.9 min | ✅ ok | 18/18 |
| G2 西虹市首富 (Comedy) | 4.45 GB | ~38 min | ✅ ok | 18/18 |

**v0.4.22+ EP3 跨片分布验证完成** — 一部史剧 + 一部喜剧，输出模式完全一致：

1. **scene detection 都成功**, 18 segments, qa.ok=true
2. **embedding top-K 重排都 100% 命中**, 无 top-1/heuristic 退化
3. **score 相似范围**（0.66-0.74），无 score 退化警报
4. **footage_coverage=1.0**, 有源片环境下画面覆盖率 100%

### ⚠️ Style/Prompt 副作用（driver config bug）

`pick_defaulted()` 第 43 行的隐式 fallback: cli 值=="热血搞笑" == `_STYLE_DEFAULT`, 让 yaml="历史悬疑" 覆盖生效。结果 LLM 按"历史悬疑"生成, 输出偏史剧基调 (虽然最终仍是中文 13 句批次)。

**影响**: 与西虹市喜剧风格不匹配, 但不影响引擎判定。**根本修复**: yaml style 应与 movie genre 同步, 或 driver 不复用 G1 yaml。

### ⚠️ 总预算：50.6 min wall-clock

G2 vs G1 阶段耗时对比:

| 阶段 | G1 满江红 | G2 西虹市首富 | 备注 |
|---|---|---|---|
| preflight + resolve + assets | < 1s | < 1s | |
| generate_script | 13.7s | 19.9s | qwen3-next |
| export_script_md | 10ms | 1ms | |
| **generate_voice** (MiMo TTS) | 9.5s | 8.4s | 18 段 × 0.5-1s |
| align_audio (faster_whisper) | **1.0 min** | **34.3s** | G1 满江红 60s vs G2 西虹市 51s |
| **detect_scenes (PySceneDetect)** | **~10.9 min** | **~38 min** | G2 是 G1 大小的 2.9 倍, 帧数近 G1 但耗时随运行实例密集度变 |
| match_clips (EP3 topk) | < 1 min | < 1 min | 18 段 embedding |
| mix_bgm + subtitle + render_video + qa | < 2 min | < 2 min | |
| **Total** | **16.3 min** | **50.6 min** | G2 三倍耗时 |

主要瓶颈 = **PySceneDetect 扫整个 ~100 minute 电影**, 与 narration 时长完全不一致。G1 满江红是 G2 西虹市 (~1/3 大小) 但耗时只有 1/5, 说明 PySceneDetect 帧率不严格线性于总帧数。

### ⚠️ HEVC 警告

`render_video` 期间 ffmpeg 报 `Could not find ref with POC 1528` 等等 G1 没出现但 G2 出现的错误（西虹市首富源片是 H.265 编码, 满江红可能是 H.264）。这些警告**不影响 final.mp4 输出**——可能是 ffmpeg 在处理参考帧时的丢失警告, 但没有导致编码失败。

实际输出 `final.mp4` 完整可播 (size 34 MB, duration 51s, has_video+has_audio=true)。
